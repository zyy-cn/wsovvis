# WSOVVIS Clean Automation Kit

This kit is a self-contained replacement for the old workflow control layer.

It assumes:
- a clean WSOVVIS code repository already exists,
- current code, tools, tests, and configs remain in place,
- old workflow docs do not need to be kept for automation to function.

Deploy by unpacking this kit at the repository root.
Then start Codex from the repo root and follow `START_AUTOMATION.md`.
