# S3a — Semantic Carrier Validity Gate

## Scientific question
Does `z_tau` function as a valid primary semantic carrier, rather than merely as a cached high-dimensional feature?

## Scope
This gate validates semantic separability and carrier necessity.
It does not validate objectness; that is `S3b`.

## Required direction
- define a semantic-carrier necessity comparator
- require class-level separability / retrieval evidence
- require at least one worked example and one visualization
